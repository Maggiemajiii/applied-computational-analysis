2 Lecture 1

2.1 1. Introduction: Themes, Concepts, and Examples

Mathematical problems can be approximately solved with computers. Most problems do not have explicit solution formulas.

Example:

  • Find roots of: \(x^5 + x^4 + 3x^2 + 2x + 2 = 0\).

  • Abel’s Impossibility Theorem: There is no algebraic solution (explicit solution formula) for general polynomial equations of degree five or higher with arbitrary constants.

  • Most partial differential equations (PDEs) have no explicit solution formula.

2.1.1 To study the approximation:

  1. Theory (Analysis):
    • Convergence: Limits of sequences that approach the true solution.
    • Finite dimensional space for approximation.
    • Stability.
  2. Application:
    • Derivation of numerical methods.
    • Interpretation of results / measure the error.
    • Adapting methods to get desired properties.
  3. Implementation:
    • Matlab, Python, C.

2.2 2. Floating Point Arithmetic

Digital computers use a finite number of bits to represent a real number.

  • \(F\): The set of Floating point numbers.

  • The position of decimal (or binary) point is stored separately from the digits.

Form of elements in F: \[x = \pm (m / \beta^t) \beta^e\]

Where:

  • \(\beta > 2\): Base (an integer, given by computer).

  • \(t \ge 1\): Precision (an integer, given by computer). e.g., 24 for single precision, 53 for double precision.

  • \(e\): Exponent (integer). Bounds \(e_{min} \le e \le e_{max}\) given by computer.

  • \(m\): Mantissa (integer), \(\beta^{t-1} \le m \le \beta^t - 1\).

2.2.1 Example: Double Precision (\(t=53, \beta=2\))

Consider the interval \((1, 2]\). Let \(e=1\): \[2^{52} \le m \le 2^{53} - 1\]

The numbers in this range are: \[1, 1+2^{-52}, 1+2 \times 2^{-52}, ..., 2 - 2^{-52}\]

  • Smallest mantissa: \(m = 2^{52}\).

  • Largest mantissa: \(m = 2^{53} - 1\).

  • Next number (let \(e=2\)): \(x = \frac{2^{52}}{2^{53}} \cdot 2^2 = 2\).

  • \(x_{max} = \frac{2^{53}-1}{2^{53}} 2^1 \approx 2 - 2^{-52}\) (approx).

(See textbook section 2.5 for details).

2.2.2 Machine Epsilon (\(\epsilon_{machine}\))

How large is the gap between numbers in \(F\)?

Define machine epsilon: \[\epsilon_{machine} = \frac{1}{2} \beta^{1-t}\]

For all \(x \in \mathbb{R}\), there exists \(x' \in F\) such that: \[|x - x'| \le \epsilon_{machine} |x|\]

Let \(fl: \mathbb{R} \to F\) denote the function giving the closest floating point approximation of a real number. For \(x \in \mathbb{R}\), with \(|\epsilon| < \epsilon_{machine}\): \[fl(x) = x(1+\epsilon)\]

For double precision: \[\epsilon_{machine} = \frac{1}{2} \cdot 2^{1-53} = 2^{-52} \approx 1.1 \times 10^{-16}\]

2.2.3 Accumulation of Error

When you compute with floating numbers: \[y = \sum_{n=1}^{1000} \frac{(-1)^n}{2n+1}\]

  • This involves 1000 additions.

  • Relative error: \(1000 \times \epsilon_{machine} \approx 10^{-13}\).

  • Need to pay attention when designing algorithms so that errors are not amplified to the point of spoiling the solution.

2.3 3. Condition / Sensitivity

Consider a problem with an input \(x\) and an output \(f(x)\).

When \(x\) is changed by an amount \(\delta x\) (of size \(||\delta x|| \le \epsilon\)), the output becomes \(f(x+\delta x)\).

The difference is: \[\delta f = f(x + \delta x) - f(x)\]

2.3.1 Definitions

  • Well-conditioned: A problem is called well-conditioned if small changes in the input lead to small changes in the output.

  • Ill-conditioned: If it is sensitive to the change of \(\delta x\), then the problem is called ill-conditioned.

2.3.2 Relative Condition Number (\(\kappa\))

How to measure the condition of a problem?

\[K = \lim_{\epsilon \to 0} \sup_{||\delta x|| \le \epsilon} \frac{||\delta f|| / ||f(x)||} {||\delta x|| / ||x||}\]

Using the Jacobian \(J = f'\): \[\kappa = \frac{||J|| \cdot ||x||}{||f(x)||}\]

  • A problem is well-conditioned if \(\kappa\) is small (e.g., \(1, 10, 10^2...\)).

  • A problem is ill-conditioned if \(\kappa\) is large (e.g., \(10^6, 10^7...\)).

Remark: For \(f: \mathbb{R}^m \to \mathbb{R}^n\), use \(||\cdot||\) for norm of \(x\) and norm of \(f(x)\). (Check textbook 1.10 for def of norms).

2.3.3 Examples

1. Consider \(f(x) = \sqrt{x}\) for \(x > 0\) \[J = f'(x) = \frac{1}{2} x^{-1/2}\]

\[\kappa = \frac{||J|| \cdot ||x||}{||f(x)||} = \frac{\frac{1}{2}x^{-1/2} \cdot x}{\sqrt{x}} = \frac{1}{2}\]

  • \(\kappa = 1/2\) is small.

  • This is a well-conditioned problem.

2. Consider \(f(x) = x_1 - x_2\), where \(x = [x_1, x_2]^T\) \(f: \mathbb{R}^2 \to \mathbb{R}\) \(J = [1, -1]\)

Let’s consider the condition number w.r.t. \(\infty\)-norm. \[\kappa = \frac{||J||_\infty}{||f||_\infty / ||x||_\infty} = \frac{2 \cdot ||x||_\infty}{|x_1 - x_2|}\]

Details:

  • \(||J||_\infty = \sup_{||y||_\infty=1} ||Jy|| = \sup |y_1 - y_2| = 2\).

  • \(||x||_\infty = \max \{ |x_1|, |x_2| \}\).

Take \(x_1 = 1, x_2 = -1 \implies ||x||_\infty = 1\). Then \(\kappa\) is reasonable.

However, when \(x_1\) and \(x_2\) are very close (\(x_1 \approx x_2\)), the denominator \(|x_1 - x_2|\) is small, so \(\kappa\) can be very large.

“Cancellation Error”: We should avoid subtraction of very close vectors in designing algorithms.

3 Lecture 2: Condition Numbers & Interpolation

3.1 1. Condition of Matrix-Vector Multiplication

Consider the function \(f(\vec{x}) = A\vec{x}\), where \(A \in \mathbb{R}^{m \times n}\).

The condition number \(\kappa\) is defined by: \[\kappa = \lim_{\epsilon \to 0} \sup_{||\delta \vec{x}|| \le \epsilon} \frac{||\delta f|| / ||f(\vec{x})||}{||\delta \vec{x}|| / ||\vec{x}||}\] \[= ||A|| \frac{||\vec{x}||}{||A\vec{x}||}\]

  • Case 1: If \(A\) is square and nonsingular: \[\frac{||\vec{x}||}{||A\vec{x}||} \le ||A^{-1}||\] Thus, \(\kappa \le ||A|| \cdot ||A^{-1}||\).

  • Case 2: For a matrix \(A \in \mathbb{R}^{m \times n}\) where \(m \ne n\), use the pseudo-inverse \(A^\dagger\) (defined with SVD): \[\kappa \le ||A|| \cdot ||A^\dagger||\]

Using the 2-norm:

  • \(||A||_2 = \sigma_1\) (largest singular value).

  • \(||A^{-1}||_2 = \frac{1}{\sigma_m}\) (smallest singular value).

  • Therefore, the condition number in 2-norm is: \[\kappa(A) = \frac{\sigma_1}{\sigma_m}\]

Remark: The condition of the problem is inherent to the problem itself. A correct algorithm would likely inherently have the same sensitivity. It is hard to solve ill-conditioned problems numerically.

Problem Comparison:

  1. Map \(\vec{x} \mapsto A\vec{x}\): \(\kappa(A) = ||A|| \cdot ||A^{-1}||\).

  2. Map \(A\vec{x} \mapsto \vec{x}\) (solving linear system): \(\kappa = \kappa(A^{-1}) = ||A^{-1}|| \cdot ||A||\).

    • If \(A\) is singular, then \(\kappa(A) = \infty\).

3.2 2. Numerical Instability

Example: Compute \(\sqrt{2}\) (approx 1.4142…).

Method 1: Babylonian Method (Stable)

  • Initialize \(x_0\).

  • Update: \(x_{k+1} = \frac{x_k}{2} + \frac{1}{x_k}\).

  • Assume it converges to \(x^*\): \(x^* = \frac{x^*}{2} + \frac{1}{x^*} \Rightarrow (x^*)^2 = 2\).

Method 2: Unstable Method

  • Start with \((x^*)^2 = 2 \Rightarrow x^* = \sqrt{2}\).

  • Rearrange: \(x^* = x^* - ( (x^*)^2 - 2 )\).

  • Iterative scheme: \(x_{k+1} = (x_k^2 - 2) + x_k\).

  • Result: This is unstable with respect to the initial guess. A small change in \(x_0\) causes a large difference, and error grows exponentially.

Definition: An algorithm that amplifies error too much is called numerically unstable.

3.3 3. Polynomial Interpolation

(Textbook 8.1 and 8.2).

Goal: Approximate function \(f\) by a “simple function” \(g\) (from a subspace).

  1. How to choose \(g\)? Polynomials, trigonometric functions, etc.

  2. How to measure error?

    • \(L^2\) norm: \(||f-g||_{L^2[a,b]} = [\int_a^b (f(x)-g(x))^2 dx]^{1/2}\).

    • \(L^\infty\) norm: \(||f-g||_{L^\infty[a,b]} = \max_{x \in [a,b]} |f(x)-g(x)|\).

Weierstrass Approximation Theorem:

Suppose \(f\) is a continuous real-valued function defined on \([a, b]\). For every \(\epsilon > 0\), there exists a polynomial \(p\) such that for all \(x \in [a, b]\): \[|f(x) - p(x)| < \epsilon\]

Equivalently: \(||f-p||_{L^\infty[a,b]} < \epsilon\).

3.4 4. Interpolation Problem & Lagrange Form

Problem Statement:

Given \((n+1)\) pairs \((x_i, y_i)\), find a polynomial \(\Pi_n \in \mathbb{P}_n\) (interpolating polynomial) such that: \[\Pi_n(x_i) = y_i \quad \text{for } i = 0, 1, ..., n\]

  • \(x_i\) are called interpolation nodes.

  • Theorem 8.1: Given distinct nodes, there exists a unique polynomial \(\Pi_n \in \mathbb{P}_n\) satisfying the condition.

Lagrange Form:

Can we find a basis of \(\mathbb{P}_n\) that leads to a simpler form?

Define Lagrange basis (Characteristic polynomials) \(l_i(x)\): \[l_i(x) = \prod_{j=0, j \ne i}^{n} \frac{x - x_j}{x_i - x_j}\]

  • Property: \(l_i(x_j) = \delta_{ij}\) (1 if \(i=j\), 0 if \(i \ne j\)).

  • Interpolant: \[\Pi_n(x) = \sum_{i=0}^n y_i l_i(x)\]

Alternate form using Nodal Polynomial:

Let \(w_{n+1}(x) = \prod_{i=0}^n (x - x_i)\) be the “nodal polynomial”. \[\Pi_n(x) = \sum_{i=0}^n \frac{w_{n+1}(x)}{(x-x_i) w'_{n+1}(x_i)} y_i\]

3.5 5. Interpolation Error

Theorem 8.2:

Let \(x_0, x_1, \dots, x_n\) be \((n+1)\) distinct nodes and \(x\) be a point in the domain of \(f\). Assume \(f \in C^{n+1}(I_x)\), where \(I_x\) is the smallest interval containing the nodes and \(x\). The interpolation error is:

\[E_n(x) = f(x) - \Pi_n f(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!} w_{n+1}(x)\]

where \(\xi \in I_x\) and \(w_{n+1}(x) = \prod_{i=0}^{n}(x - x_i)\) is the nodal polynomial of degree \(n+1\).

3.5.1 Comparison between Taylor vs. Interpolation

Both methods provide polynomial approximations of \(f(x) \in \mathbb{P}_n\), but they utilize different information:

  • Taylor Polynomial: Uses information from one point (the point \(x_0\) and its derivatives).

  • Interpolation Polynomial: Uses information from \((n+1)\) distinct points.

We seek an error bound where \(\max |E_n(x)|\) is minimized. If \(M = \max_{x \in [a,b]} |f^{(n+1)}(x)|\), then: \[\max_{x \in [a,b]} |E_n(x)| \le \frac{M}{(n+1)!} \max_{x \in [a,b]} |w_{n+1}(x)|\]

  • Example 1 (Convergence): \(f(x) = e^x\) on \([0, 1]\) with equally spaced nodes.
    • \(M \le e\).
    • Using Stirling’s approximation, \(|w_n(x)| \le (n+1)! (\frac{1}{n})^{n+1} \approx C\sqrt{n}e^{-n}\).
    • The error decays as \(n\) increases.
  • Example 2 (Divergence - Runge’s Counter Example): \(f(x) = \frac{1}{1+25x^2}\) on \([-1, 1]\) with equally spaced nodes.
    • As \(n \to \infty\), the maximum error grows exponentially.
    • While \(|w_{n+1}(x)|\) decays, the term \(\frac{f^{(n+1)}(\xi)}{(n+1)!}\) grows much faster.
    • Conclusion: High-degree interpolants with equally spaced nodes can be “disastrous.”

3.6 6. Newton Form of Interpolation

Motivation

In the Lagrange representation, adding a new node or collecting an additional data point requires changing every term in the formula. To improve efficiency, we use the Newton form, which allows for adding points at a much lower computational cost.

Recursive Structure

The interpolating polynomial of degree \(n\) is built by adding a correction term to the previous polynomial of degree \(n-1\): \[\Pi_n(x) = \Pi_{n-1}(x) + q_n(x)\] To ensure that \(\Pi_n(x)\) still passes through all previous points \(x_0, \dots, x_{n-1}\), the correction term \(q_n(x)\) must satisfy:

  • \(q_n \in \mathbb{P}_n\) (it is a polynomial of degree at most \(n\))

  • \(q_n(x_i) = 0\) for \(i = 0, 1, \dots, n-1\)

Nodal Polynomials and Coefficients

Since \(q_n\) has roots at \(x_0, \dots, x_{n-1}\), it must be a multiple of the nodal polynomial \(w_n(x)\). We define this as: \[q_n(x) = c_n w_n(x)\] where \(w_n(x) = \prod_{i=0}^{n-1}(x-x_i)\). To find the specific coefficient \(c_n\), we evaluate the polynomial at the new node \(x_n\): \[c_n = \frac{q_n(x_n)}{w_n(x_n)} = \frac{f(x_n) - \Pi_{n-1}(x_n)}{w_n(x_n)}\]

The Complete Newton Form

By applying this recursive logic from \(c_0\) up to \(c_n\), we arrive at the full representation: \[\Pi_n(x) = c_0 + c_1(x-x_0) + c_2(x-x_0)(x-x_1) + \dots + c_n(x-x_0)(x-x_1)\dots(x-x_{n-1})\] In compact summation notation: \[\Pi_n(x) = \sum_{j=0}^{n} c_j \prod_{i=0}^{j-1} (x-x_i)\]

Computational Efficiency

Adding a new node \(x_{n+1}\) is simple because we only need to compute one additional coefficient \(c_{n+1}\) and the next nodal polynomial \(w_{n+1}(x)\). The existing terms of \(\Pi_n(x)\) remain unchanged. These coefficients \(c_n\) are typically computed using a divided differences table to maximize efficiency.

4 Lecture 3: Newton Form, Error Analysis, and Spline Interpolation

4.1 1. Newton Form of Interpolation (Recap & Details)

4.1.1 Newton Form Structure

Recall that adding points to the Lagrange form is computationally expensive. The Newton Form allows for efficient updates when new nodes are added.

The polynomial is written as: \[\Pi_n(x) = c_0 + c_1(x-x_0) + c_2(x-x_0)(x-x_1) + \dots + c_n \prod_{i=0}^{n-1}(x-x_i)\]

This can be written compactly as: \[\Pi_n(x) = \sum_{j=0}^{n} c_j \prod_{i=0}^{j-1}(x-x_i)\]

4.1.2 Divided Differences

The coefficients \(c_j\) are called the \(n\)-th Newton divided differences. Notation: \(c_j = f[x_0, x_1, \dots, x_j]\).

Recursive Definition:

  • 0-th order: \(c_0 = f[x_0] = f(x_0) = y_0\).

  • 1-st order: \(c_1 = f[x_0, x_1] = \frac{f[x_1] - f[x_0]}{x_1 - x_0}\).

  • 2-nd order: \(c_2 = f[x_0, x_1, x_2] = \frac{f[x_1, x_2] - f[x_0, x_1]}{x_2 - x_0}\).

  • General Recursive Formula: \[c_n = f[x_0, \dots, x_n] = \frac{f[x_1, \dots, x_n] - f[x_0, \dots, x_{n-1}]}{x_n - x_0}\]

4.1.3 Example

Exact function: \(f(x) = x^3 - 2x^2 + 1\). Nodes: \(x_i = 0, 1, 2, 3\). Values:

  • \(f(0) = 1\)

  • \(f(1) = 0\)

  • \(f(2) = 1\)

  • \(f(3) = 10\)

By computing the divided difference table, the polynomial is: \[\Pi_3(x) = 1 - x + x(x-1) + x(x-1)(x-2)\] \[= 1 - x + x^2 - x + x(x^2 - 3x + 2)\] (Simplifies back to original function).

4.2 2. Interpolation Error Analysis

Theorem: Let \(x_0, \dots, x_n\) be distinct nodes. Assume \(f \in C^{n+1}(I_x)\). The error is given by: \[E_n(x) = f(x) - \Pi_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!} w_{n+1}(x)\] Where \(w_{n+1}(x)\) is the nodal polynomial: \[w_{n+1}(x) = \prod_{i=0}^{n} (x - x_i)\]

Error Bound: \[|E_n(x)| \le \frac{M}{(n+1)!} \max |w_{n+1}(x)|\] Where \(M = \max |f^{(n+1)}(x)|\).

4.2.1 The Problem with High-Degree Interpolation

Question: Does increasing the number of nodes always reduce error?

Example 1: \(f(x) = e^x\) on \([0, 1]\).

  • Error decreases as \(n\) increases (well-behaved).

Example 2 (Runge’s Phenomenon): \[f(x) = \frac{1}{1+25x^2}, \quad x \in [-1, 1]\]

  • With equally spaced nodes, the maximum error grows exponentially as \(n \to \infty\).

  • High-degree polynomials oscillate wildly near the endpoints.

Solution: Chebyshev Interpolation

Instead of equally spaced nodes, we choose nodes to minimize \(\max |w_{n+1}(x)|\). \[x_j = \cos \left( \frac{2j+1}{2n+2} \pi \right)\]

  • Remark: There is no universal optimal choice of nodes for all functions (Faber’s Theorem), but Chebyshev is often much better than equidistant.

4.3 3. Spline Interpolation

Given the issues with high-degree polynomials (Runge’s phenomenon), we use a different approach: Use low-degree polynomials in sub-intervals.

4.3.1 Definition: Spline of Order \(k\)

Given nodes \(a = x_0 < x_1 < \dots < x_n = b\). A function \(S_k\) is a spline of order \(k\) if:

  1. Piecewise Polynomial: On each sub-interval \([x_j, x_{j+1}]\), \(S_k\) is a polynomial of degree \(\le k\).

  2. Smoothness: \(S_k \in C^{k-1}[a, b]\) (Continuous up to the \((k-1)\)-th derivative).

4.4 4. Linear Splines (\(k=1\))

Connect the data points with straight lines.

  • Properties:
    • \(S_1\) is a polynomial of degree 1 on each interval.
    • \(S_1 \in C^0[a, b]\) (Continuous, but derivative is discontinuous).
    • \(S_1(x_j) = f(x_j)\).
  • Error Analysis: Let \(h_j = x_{j+1} - x_j\) (interval width). \[|S_1(x) - f(x)| \le \frac{1}{4} \max |f''(x)| h_{max}^2\]
    • As the max spacing \(h_{max} \to 0\), the error converges to 0 with order \(O(h^2)\).

4.5 5. Cubic Splines (\(k=3\))

We often want a smoother approximation (e.g., \(C^2\) continuity).

  • Properties:
    1. Restricted to \([x_j, x_{j+1}]\), it is a polynomial of degree 3 (\(P_i \in \mathbb{P}_3\)).
    2. \(S_3 \in C^2[a, b]\) (Function, 1st deriv, and 2nd deriv are continuous).
    3. \(S_3(x_j) = f(x_j)\) (Interpolation condition).

4.5.1 Deriving the System

Let \(P_i(x)\) be the cubic polynomial on interval \([x_i, x_{i+1}]\). \[P_i(x) = a_i + b_i(x-x_i) + c_i(x-x_i)^2 + d_i(x-x_i)^3\]

Unknowns: \(4n\) unknowns (since there are \(n\) intervals).

Constraints:

  1. Interpolation: \(P_i(x_i) = a_i = f(x_i)\). (Determines \(a_i\)).

  2. Continuity (\(C^0\)): \(P_i(x_{i+1}) = P_{i+1}(x_{i+1})\). \[a_i + b_i h_i + c_i h_i^2 + d_i h_i^3 = a_{i+1}\]

  3. Smoothness (\(C^1\)): \(P'_i(x_{i+1}) = P'_{i+1}(x_{i+1})\). \[b_i + 2c_i h_i + 3d_i h_i^2 = b_{i+1}\]

  4. Curvature (\(C^2\)): \(P''_i(x_{i+1}) = P''_{i+1}(x_{i+1})\). \[2c_i + 6d_i h_i = 2c_{i+1} \implies c_i + 3d_i h_i = c_{i+1}\]

4.5.2 Boundary Conditions

The internal conditions give \((3n-2)\) equations for \(3n\) unknowns. We need 2 more conditions at the endpoints (\(x_0\) and \(x_n\)).

Natural Spline Boundary: \[S''(a) = S''(b) = 0\]

4.5.3 Solving

The system can be formulated as a matrix equation: \[A \vec{c} = \vec{r}\] Where \(\vec{c}\) contains the unknown coefficients.

  • The matrix \(A\) is Tridiagonal.

  • This system can be solved very efficiently (linear time, \(O(n)\)) using algorithms like the Thomas algorithm.

5 Lecture 4: Numerical Differentiation

5.1 1. Introduction

We consider the numerical approximation of the differentiation of a function \(f\), given values of \(f\) at a set of points (a grid).

5.1.1 Concepts of Accuracy

  1. Degree of Accuracy (\(k\)): If an approximation formula has zero error for polynomials of degree \(k\), we say it has a degree of accuracy \(k\).

  2. Order of Accuracy (\(p\)): The approximation has order of accuracy \(p\) if the error scales like \(O(h^p)\) as \(h \to 0\), where \(h\) denotes the grid spacing (interval width).

5.1.2 Stencils

The stencil is the set of points used in the approximation.

  • Forward: Uses points \(\ge x\) (e.g., \(x, x+h\)).

  • Backward: Uses points \(\le x\) (e.g., \(x, x-h\)).

  • Central: Uses points surrounding \(x\) (e.g., \(x-h, x, x+h\)).

5.2 2. Method 1: Polynomial Interpolation

This method fits a polynomial to the data points and differentiates it.

Recall Lagrange Form: \[\Pi_n f(x) = \sum_{i=0}^n f_i l_i(x)\] where \(l_i(x) = \prod_{j \neq i} \frac{x - x_j}{x_i - x_j}\).

By taking the derivative of the interpolant: \[f'(x_k) \approx \sum_{i=0}^n f_i l_i'(x_k) = \sum_{i=0}^n c_i f_i\] where the weights are \(c_i = l_i'(x_k)\).

5.2.1 Example: Central Difference (3 points)

We want to approximate \(f'(x_1)\) (or \(f'(0)\)) using the stencil \(\{-h, 0, h\}\).

1. Define Lagrange Basis: \[ \begin{aligned} l_0(x) &= \frac{(x-0)(x-h)}{(-h-0)(-h-h)} = \frac{x(x-h)}{2h^2} \\ l_1(x) &= \frac{(x+h)(x-h)}{(0+h)(0-h)} = \frac{-(x+h)(x-h)}{h^2} \\ l_2(x) &= \frac{(x+h)(x-0)}{(h+h)(h-0)} = \frac{x(x+h)}{2h^2} \end{aligned} \]

2. Differentiate and Evaluate at \(x_1 = 0\): \[ \begin{aligned} c_0 &= l_0'(0) = -\frac{1}{2h} \\ c_1 &= l_1'(0) = 0 \\ c_2 &= l_2'(0) = \frac{1}{2h} \end{aligned} \]

3. Final Formula: \[f'(0) \approx -\frac{1}{2h}f(-h) + 0 \cdot f(0) + \frac{1}{2h}f(h) = \frac{f(h) - f(-h)}{2h}\]

Accuracy: This central difference formula has Order \(O(h^2)\) and Degree 2.

For a general stencil with \((n+1)\) points, the order of accuracy for differentiation is generally \(n\).

5.3 3. Method 2: Taylor Series Expansion

Interpolation does not generalize well to higher-order derivatives. Instead, we use Taylor series expansions with undetermined coefficients.

5.3.1 Example A: Central Difference for \(f''(x)\)

We use the stencil \(\{x-h, x, x+h\}\) to approximate \(f''(x)\).

Ansatz: \[\delta^2 f = \frac{a f(x-h) + b f(x) + c f(x+h)}{h^2}\]

Taylor Expansion: \[f(x \pm h) = f(x) \pm hf'(x) + \frac{h^2}{2}f''(x) \pm \frac{h^3}{6}f^{(3)}(x) + \frac{h^4}{24}f^{(4)}(\xi)\]

Plugging these into the ansatz, we set up a system of linear equations to match coefficients:

  1. \(a + b + c = 0\) (coefficient of \(f(x)\))

  2. \(c - a = 0\) (coefficient of \(f'(x)\))

  3. \(\frac{1}{2}(c + a) = 1\) (coefficient of \(f''(x)\))

Solution: \(a=1, b=-2, c=1\).

Resulting Formula: \[f''(x) \approx \frac{f(x-h) - 2f(x) + f(x+h)}{h^2}\]

Error Analysis: \[E(x) = \frac{h^2}{24}[f^{(4)}(\xi_{-}) + f^{(4)}(\xi_{+})] = \frac{h^2}{12}f^{(4)}(\xi)\] Due to the symmetry of the central difference, odd error terms cancel out, giving us one extra order of accuracy for free (\(O(h^2)\)).

5.3.2 Example B: Backward Difference for \(f'(x)\) (Extrapolation)

We use the stencil \(\{x-2h, x-h, x\}\) to approximate \(f'(x)\).

Ansatz: \[D(h) = \frac{a f(x) + b f(x-h) + c f(x-2h)}{h}\]

Taylor Expansion: \[ \begin{aligned} f(x-h) &= f(x) - hf'(x) + \frac{h^2}{2}f''(x) - \dots \\ f(x-2h) &= f(x) - 2hf'(x) + 2h^2f''(x) - \dots \end{aligned} \]

System of Equations:

  1. \(a + b + c = 0\)

  2. \(-b - 2c = 1\)

  3. \(\frac{1}{2}b + 2c = 0\)

Solution: \(a = 3/2, b = -2, c = 1/2\).

Resulting Formula: \[f'(x) \approx \frac{\frac{3}{2}f(x) - 2f(x-h) + \frac{1}{2}f(x-2h)}{h}\]

6.1 Richardson Extrapolation

The core idea of Richardson extrapolation is to improve the accuracy of a numerical formula and estimate the error by canceling out the leading terms of an asymptotic expansion.

6.1.1 Theoretical Foundation

Recall the Taylor series expansion for a function \(f(x)\): \[f(x) = T_{n}(x) + \frac{f^{(n+1)}(\xi)}{(n+1)!}(x - x_{0})^{n+1}\] The error can be expressed as an asymptotic series: \[E(x) = C_{n+1}(x - x_{0})^{n+1} + C_{n+2}(x - x_{0})^{n+2} + \dots\] where the coefficients \(C_j\) are known. Since each term is asymptotically smaller than the previous one as \(x \to x_0\), we can combine different step sizes to cancel the largest error terms.

6.1.2 Application to Derivatives

Consider the central difference approximation for \(L = f'(x)\): \[A(h) = \frac{f(x+h) - f(x-h)}{2h}\] The Taylor expansion shows that: \[A(h) = L + C_2 h^2 + C_4 h^4 + C_6 h^6 + \dots\]

If we compute \(A\) with step size \(h\) and \(h/2\), we can eliminate the \(h^2\) term: \[A_1(h) = \frac{4A(\frac{h}{2}) - A(h)}{3} = L + O(h^4)\]

6.1.3 General Iteration Formula

Iterating this process yields a table of approximations where \(A_k(h) = L + O(h^{2k})\): \[A_k(h) = \frac{1}{4^{k}-1} \left( 4^{k} A_{k-1}\left(\frac{h}{2}\right) - A_{k-1}(h) \right)\]

6.2 Numerical Integration (Newton-Cotes)

Numerical integration approximates \(\int_{a}^{b} f(x) dx\) by integrating a Lagrange interpolating polynomial \(P_n(x)\) constructed using equally spaced nodes.

6.2.1 Coordinate Transformation

To simplify, we often integrate over \(s \in [0, 1]\) and then transform back to \(x \in [a, b]\) using:

  • \(x = a + (b-a)s = a + hs\)

  • \(dx = (b-a)ds = h ds\)

6.2.2 1. Trapezoidal Rule (Closed, \(n=1\))

Uses nodes at the endpoints \(s_0=0\) and \(s_1=1\). \[\int_{a}^{b} f(x) dx \approx \frac{h}{2} [f(a) + f(b)]\] Error term: \(-\frac{h^3}{12} f''(\xi) = O(h^3)\)

6.2.3 2. Midpoint Rule (Open, \(n=0\))

Uses the interior node \(s_0 = 1/2\). \[\int_{a}^{b} f(x) dx \approx (b-a) f\left(\frac{a+b}{2}\right)\] Error term: \(\frac{(b-a)^3}{24} f''(\xi)\)

6.2.4 Closed Newton-Cotes Weights (\(I_n(f) = h \sum w_i f_i\))

2 Simpson’s \(1/3\) \(4/3\) \(1/3\)

6.2.5 Error Characteristics

  • Degree of Exactness: The highest degree of polynomial for which the rule is exact.

  • For even \(n\): Degree of exactness is \(n+1\); Error is \(O(h^{n+3})\).

  • For odd \(n\): Degree of exactness is \(n\); Error is \(O(h^{n+2})\).

6.3 Composite Formulas

To integrate over a large interval, we divide \([a, b]\) into \(n\) sub-intervals and apply low-degree formulas to each.

6.3.1 Composite Trapezoidal Rule

\[\int_{a}^{b} f(x) dx \approx \frac{h}{2} \left[ f(x_0) + 2\sum_{k=1}^{n-1} f(x_k) + f(x_n) \right]\]

Total Error: \(|E_T| \le \frac{M(b-a)h^2}{12}\) where \(M = \max |f''(x)|\).

6.3.2 Composite Simpson’s Rule

Divides the interval into an even number of sub-intervals.

Total Error: \(|E_S| \le \frac{M(b-a)h^4}{180}\) where \(M = \max |f^{(4)}(x)|\).

7 Lecture 6: Approximation with Orthogonal Bases

Previously, we discussed interpolation, which uses polynomials to approximate a function given values at specific points. However, if we have more comprehensive information about a function \(f\), we might want to approximate \(f\) within a space of simple functions using orthogonal bases. This approach is highly convenient for both mathematical analysis and computation.

For an interval \([a, b]\), we seek a basis \(\{\Phi_j\}\) such that the first \(n\) terms can be used to accurately approximate the function.

7.1 1. The Weighted \(L^2\) Norm and Inner Product

Let \(w(x) > 0\) be a nonnegative integrable weight function on the interval \([a, b]\). For a complex-valued function \(f\) on \([a, b]\), we define the weighted \(L^2\) norm:

\[||f||_w = \left(\int_a^b |f(x)|^2 w(x) dx\right)^{\frac{1}{2}}\]

This norm has an associated inner product (where \(\overline{g}(x)\) denotes the complex conjugate):

\[\langle f, g \rangle_w = \int_a^b f(x)\overline{g}(x) w(x) dx\]

We consider approximating functions within the weighted \(L^2\) space:

\[L_w^2([a,b]) = \left\{ f: [a,b] \rightarrow \mathbb{C} \text{ s.t. } \int_a^b |f(x)|^2 w(x) dx < \infty \right\}\]

A basis \(\{\Phi_j\}\) (for \(j = 1, 2, \dots\)) for \(L_w^2([a,b])\) is a set of functions such that any \(f\) in the space can be uniquely expressed as a series:

\[f = \sum_{j=1}^\infty c_j \Phi_j\]

This convergence is in the sense of the \(||\cdot||_w\) norm:

\[\lim_{N \rightarrow \infty} \left|\left| f - \sum_{j=1}^N c_j \Phi_j \right|\right|_w = 0\]

A set of functions \(\{\Phi_j\}\) is called orthogonal with respect to \(w\) if \(\langle \Phi_i, \Phi_j \rangle_w = 0\) for \(i \neq j\).

7.2 2. Properties of Orthogonality

Given an orthogonal basis \(\{\Phi_j\}\) of \(L_w^2([a,b])\), we have several useful properties:

7.2.1 1. Coefficient Computation

The coefficients \(c_k\) in the representation \(f = \sum_{j=1}^\infty c_j \Phi_j\) can be found by taking the inner product of \(f\) with \(\Phi_k\):

\[\langle f, \Phi_k \rangle_w = \left\langle \sum_{j=1}^\infty c_j \Phi_j, \Phi_k \right\rangle_w = c_k \langle \Phi_k, \Phi_k \rangle_w\]

Solving for \(c_k\) yields:

\[c_k = \frac{\langle f, \Phi_k \rangle_w}{\langle \Phi_k, \Phi_k \rangle_w}\]

7.2.2 2. Best Approximation Property

Let \(f_N = \sum_{j=1}^N c_j \Phi_j\) be the first \(N\) terms of the series. \(f_N\) is the best approximation of \(f\) in the subspace \(S_N = \text{span}\{\Phi_1, \Phi_2, \dots, \Phi_N\}\). That is:

\[f_N = \text{argmin}_{g \in S_N} ||f - g||_w\]

(Hint for proof: Expand \(|| \sum_{j=1}^N d_j \Phi_j - f ||_w^2\) and complete the square for the coefficients).

7.2.3 3. Approximation Error

By Parseval’s theorem, the approximation error is given by:

\[\left|\left| f - \sum_{j=1}^N c_j \Phi_j \right|\right|_w^2 = \sum_{j=N+1}^\infty c_j^2 ||\Phi_j||_w^2\]

If \(c_j\) decays at a certain rate with respect to \(j\), we can use this to estimate the corresponding approximation error.

7.3 3. Generating Orthogonal Bases

7.3.1 The Gram-Schmidt Process

Given a standard basis \(\{f_j\}\) of a function space, we can convert it into an orthogonal basis \(\{\Phi_j\}\):

  • \(\Phi_0 = f_0\)

  • \(\Phi_1 = f_1 - \frac{\langle f_1, \Phi_0 \rangle_w}{\langle \Phi_0, \Phi_0 \rangle_w} \Phi_0\)

  • \(\Phi_j = f_j - \sum_{k=0}^{j-1} \frac{\langle f_j, \Phi_k \rangle_w}{\langle \Phi_k, \Phi_k \rangle_w} \Phi_k\)

Note: We can further normalize the basis by scaling, and different scaling conventions may be used.

7.3.2 The Three-Term Recurrence (Simplification for Polynomials)

Consider the polynomial space \(\mathbb{P}_j\). We want to derive an orthogonal basis \(\{\Phi_j\}\). Suppose we have orthogonalized up to \(\Phi_{j-1}\).

Instead of applying Gram-Schmidt to \(f_{j+1} = x^{j+1}\), consider choosing \(f_{j+1} = x\Phi_j\). Because \(x\Phi_j \in \mathbb{P}_{j+1}\), we can use it to find the next orthogonal polynomial.

Recall that for \(k \le j-2\), we have \(x\Phi_k \in \mathbb{P}_{j-1}\). Therefore: \[\langle x\Phi_j, \Phi_k \rangle_w = \langle \Phi_j, x\Phi_k \rangle_w = 0 \quad \text{for } k \le j-2\]

This drastically simplifies the Gram-Schmidt process to just three terms:

\[\Phi_{j+1} = x\Phi_j - \frac{\langle x\Phi_j, \Phi_j \rangle_w}{\langle \Phi_j, \Phi_j \rangle_w} \Phi_j - \frac{\langle x\Phi_j, \Phi_{j-1} \rangle_w}{\langle \Phi_{j-1}, \Phi_{j-1} \rangle_w} \Phi_{j-1}\]

This can be written as: \[\Phi_{j+1} = (x - \alpha_j)\Phi_j - \beta_j \Phi_{j-1}\] where \(\alpha_j\) and \(\beta_j\) are constants. This is computationally much more convenient.

7.4 4. Legendre Polynomials

Consider the interval \([a,b] = [-1, 1]\) and the weight function \(w(x) = 1\). Using the Gram-Schmidt process and the three-term recurrence trick, we generate the Legendre polynomials:

  • \(\Phi_0(x) = 1\)

  • \(\Phi_1(x) = x - \frac{\langle x, 1 \rangle}{\langle 1, 1 \rangle} 1 = x\)

  • \(\Phi_2(x) = x \cdot x - \frac{\langle x^2, x \rangle}{\langle x, x \rangle} x - \frac{\langle x^2, 1 \rangle}{\langle 1, 1 \rangle} 1 = x^2 - 0 - \frac{1}{3} = x^2 - \frac{1}{3}\)

  • \(\Phi_3(x) = x(x^2 - \frac{1}{3}) - \frac{\langle x\Phi_2, \Phi_2 \rangle}{\langle \Phi_2, \Phi_2 \rangle} \Phi_2 - \frac{\langle x\Phi_2, \Phi_1 \rangle}{\langle \Phi_1, \Phi_1 \rangle} \Phi_1 = x^3 - \frac{3}{5}x\)

These satisfy \(\int_{-1}^1 \Phi_i(x) \Phi_j(x) dx = 0\) for \(i \neq j\).

Convention: Legendre polynomials are typically normalized such that \(\Phi_j(1) = 1\). By this process, the recurrence is: \((j+1)\Phi_{j+1} - (2j+1)x\Phi_j + j\Phi_{j-1} = 0\).

7.4.1 Example: Linear Approximation

Construct the best least squares line for \(f(x) = e^x\) on the interval \([0, 1]\).

  1. Map domain: Introduce \(s \in [-1, 1]\) where \(x = \frac{s+1}{2}\). Thus, \(s = 2x - 1\).

  2. Transform function: \(g(s) = f\left(\frac{s+1}{2}\right) = e^{\frac{s+1}{2}}\).

  3. Compute coefficients: Represent \(g(s) = \sum_{j=0}^\infty c_j \Phi_j(s)\). The first two terms give the best linear approximation in the least-squares sense:

    • \(c_0 = \frac{\langle g, \Phi_0 \rangle}{\langle \Phi_0, \Phi_0 \rangle} = \frac{\int_{-1}^1 e^{\frac{s+1}{2}} 1 ds}{\int_{-1}^1 1 ds} = e - 1\)

    • \(c_1 = \frac{\langle g, \Phi_1 \rangle}{\langle \Phi_1, \Phi_1 \rangle} = \dots = 9 - 3e\)

  4. Result: \(g(s) \approx (e - 1) + (9 - 3e)s\). Substituting back to \(x\): \[f(x) \approx (e - 1) + (9 - 3e)(2x - 1)\]

7.5 5. Chebyshev Polynomials

Let’s consider a different orthogonal basis for \(L_w^2([-1, 1])\) using the weight function \(w(x) = \frac{1}{\sqrt{1 - x^2}}\).

The inner product is: \[\langle f, g \rangle_w = \int_{-1}^1 \frac{f(x)g(x)}{\sqrt{1 - x^2}} dx\]

Using the substitution \(x = \cos\theta\) (where \(\theta \in [0, \pi]\)), this transforms into: \[\langle f, g \rangle_w = \int_0^\pi f(\cos\theta)g(\cos\theta) d\theta\]

We know that \(\{\cos(k\theta)\}\) is an orthogonal set on \([0, \pi]\). Let \(f(\cos\theta) = \cos(k\theta)\). Since \(\theta = \cos^{-1}(x)\), this suggests polynomials of the form: \[T_j(x) = \cos(j \cos^{-1}(x))\]

Trigonometric identities guarantee that these form valid polynomials, known as the Chebyshev polynomials.

  • \(T_0(x) = 1\)

  • \(T_1(x) = x\)

  • \(T_2(x) = \cos(2\cos^{-1}(x)) = 2x^2 - 1\)

Recurrence Relation: \(T_{j+1}(x) = 2x T_j(x) - T_{j-1}(x)\).

(Proof leverages the sum and difference trigonometric identities: \(\cos((j+1)\theta) + \cos((j-1)\theta) = 2\cos(j\theta)\cos\theta\)).

7.5.1 Chebyshev Nodes

The roots of \(T_j(x) = 0\) are the Chebyshev nodes. Setting \(\cos(j \cos^{-1}(x)) = 0\) yields: \[x_n = \cos\left(\frac{(n + \frac{1}{2})\pi}{j}\right) \quad \text{for } n = 0, 1, \dots, j-1\]

These roots cluster near the endpoints \(\pm 1\) and help produce less oscillatory interpolants.

7.5.2 Minimax Property

A monic polynomial has a leading coefficient of 1. Since \(T_{j+1} = 2x T_j - T_{j-1}\), the leading coefficient of \(T_j(x)\) is \(2^{j-1}\). Thus, the monic Chebyshev polynomials are defined as \(\tilde{T}_j = 2^{1-j} T_j\).

Theorem: The monic Chebyshev polynomials \(\tilde{T}_j\) have the minimax property. They minimize: \[\max_{x \in [-1, 1]} |p(x)| \quad \text{for monic } p \in \mathbb{P}_j\] Since \(|T_j(x)| \le 1\) and reaches 1, the minimum possible maximum for a monic polynomial of degree \(j\) on \([-1, 1]\) is exactly \(2^{1-j}\).

Equivalently, using Chebyshev nodes for interpolation minimizes the upper bound of the error term \(\prod_{k=0}^{j-1}(x - x_k)\) to \(2^{1-j}\). (Proof by contradiction: Assume a monic polynomial \(w_j\) has a smaller maximum, and evaluate \(f_j = \tilde{T}_j - w_j\) at the extrema of \(T_j\) to show it has too many roots).

7.6 6. Numerical Integration (Gaussian Quadrature)

For weighted integrals \(I = \int_a^b f(x)w(x)dx\), we previously used Newton-Cotes formulas with equally spaced nodes (degree of exactness \(n\) or \(n+1\)).

Now, we allow both the nodes \(\{x_k\}_{k=0}^n\) and the coefficients \(C_k\) to be unknowns. With \(2n+2\) unknowns, we can construct a formula with a higher degree of accuracy (\(2n+1\)).

Theorem: Let \(\{\Phi_n\}\) be a set of orthogonal polynomials with respect to \(\langle f, g \rangle_w\). Let \(x_0, x_1, \dots, x_n\) be the \((n+1)\) distinct real zeros of \(\Phi_{n+1}(x)\) in \([a,b]\). Let \(l_k(x)\) be the \(k\)-th Lagrange basis polynomial for these zeros, and let weights be: \[C_k = \int_a^b l_k(x)w(x)dx\] Then the Gaussian Quadrature formula \(\sum_{k=0}^n C_k f(x_k)\) has a degree of accuracy of \(2n+1\).

Proof Outline: Suppose \(f \in \mathbb{P}_{2n+1}\). By polynomial division, \(f(x) = q(x)\Phi_{n+1}(x) + r(x)\), where degrees of \(q\) and \(r\) are \(\le n\).

The exact integral is:

\[I = \langle q, \Phi_{n+1} \rangle_w + \int_a^b r(x)w(x)dx\]

By orthogonality, \(\langle q, \Phi_{n+1} \rangle_w = 0\).

For the approximation: \[I_{\text{approx}} = \sum_{k=0}^n C_k (q(x_k)\Phi_{n+1}(x_k) + r(x_k))\]

Since the nodes \(x_k\) are roots of \(\Phi_{n+1}\), the first term vanishes. Thus, \(I_{\text{approx}} = \sum_{k=0}^n C_k r(x_k)\). Because \(r \in \mathbb{P}_n\), the quadrature rule integrates it exactly, making \(I = I_{\text{approx}}\).

7.6.1 Convergence and Examples

Convergence holds for Gaussian quadrature as \(n \rightarrow \infty\):

\[\lim_{n \rightarrow \infty} \left| \int_{-1}^1 f(x)w(x)dx - \sum_{k=0}^n C_k f(x_k) \right| = 0\]

Example (\(n=1\), \(w(x)=1\) on \([-1, 1]\)):

We use the roots of the Legendre polynomial \(\Phi_2(x) = \frac{1}{2}(3x^2 - 1)\).

The roots are \(x_0 = -\frac{1}{\sqrt{3}}\) and \(x_1 = \frac{1}{\sqrt{3}}\).

Calculating the weights \(C_0\) and \(C_1\) via integrals of the Lagrange basis polynomials yields \(C_0 = 1\) and \(C_1 = 1\).

The approximation is: \[\int_{-1}^1 f(x)dx \approx f\left(-\frac{1}{\sqrt{3}}\right) + f\left(\frac{1}{\sqrt{3}}\right)\] This simple 2-point formula has a degree of accuracy of 3.

(Note: Standard Gaussian quadrature does not use the endpoints \(a\) and \(b\). When including the endpoints is necessary, we use Gaussian-Lobatto quadrature).

8 Lecture 7: The Discrete Fourier Transform (DFT)

The Discrete Fourier Transform is a fundamental tool in:

  • Digital signal processing
  • Image processing
  • Solving PDEs

8.1 1. Fourier Series

Consider functions in \([0, 2\pi]\). We can write:

\[f(x) = \sum_{k=-\infty}^{\infty} c_k e^{ikx}\]

The functions \(e_k(x) = e^{ikx}\) are orthogonal in the inner product:

\[\langle f, g \rangle = \int_0^{2\pi} f(x)\overline{g(x)}\, dx\]

In this section, \([0, 2\pi]\) is considered as the space of \(2\pi\)-periodic functions:

\[f(x + 2\pi) = f(x) \quad \text{for all } x \in \mathbb{R}\]

By this periodicity property, we only need to consider \(f(x)\) within one period \([0, 2\pi]\). If we require \(f\) to have continuity at the boundary, then \(f(0) = f(2\pi)\).

8.1.1 Fourier Coefficients

The Fourier coefficients are computed by:

\[c_k = \frac{1}{2\pi}\int_0^{2\pi} f(x) e^{-ikx}\, dx = \frac{\langle f, e^{ikx} \rangle}{2\pi}\]

8.1.2 Real Form of the Fourier Series

For a real-valued function \(f\), we have \(c_{-k} = \overline{c_k}\), and we can write:

\[c_k = \frac{1}{2}(a_k - ib_k), \quad c_{-k} = \frac{1}{2}(a_k + ib_k)\]

where:

\[a_k = \frac{1}{\pi}\int_0^{2\pi} f(x)\cos(kx)\, dx, \quad b_k = \frac{1}{\pi}\int_0^{2\pi} f(x)\sin(kx)\, dx\]

The Fourier series in real form is:

\[f(x) = \frac{a_0}{2} + \sum_{k=1}^{\infty}(a_k \cos(kx) + b_k \sin(kx))\]

8.1.3 Partial Sums and Best Approximation

The \(N\)-th partial sum is:

\[S_N(x) = \frac{a_0}{2} + \sum_{k=1}^{N}(a_k \cos(kx) + b_k \sin(kx)) = \sum_{k=-N}^{N} c_k e^{ikx}\]

\(S_N\) is the least-squares approximation of \(f\) in \(\text{span}\{e^{ikx}\}_{k=-N}^{N}\) in the \(L^2\) norm.

8.1.4 Examples

  • Triangle wave: \(f(x) = x\) for \(x \in (-\pi, \pi]\), extended \(2\pi\)-periodically.
  • Square wave: \(f(x) = \begin{cases} 1 & x > 0 \\ -1 & x < 0 \end{cases}\), extended \(2\pi\)-periodically.

8.2 2. Convergence of Fourier Series

8.2.1 Norm Convergence

If \(f\) is real-valued in \(L^2([-\pi, \pi])\):

\[\lim_{N\to\infty} \|f - S_N\|_2 = 0\]

8.2.2 Pointwise Convergence

For \(f \in L^1([-\pi, \pi])\), if \(f\) and \(f'\) are piecewise continuous:

\[\lim_{n\to\infty} S_n(x) = \begin{cases} f(x) & \text{if } f \text{ is continuous at } x \\[6pt] \dfrac{f(x^+) + f(x^-)}{2} & \text{if } f \text{ has a jump at } x \end{cases}\]

where \(f(x^+) = \lim_{\varepsilon \to 0^+} f(x+\varepsilon)\) and \(f(x^-) = \lim_{\varepsilon \to 0^+} f(x-\varepsilon)\).

8.2.3 Gibbs Phenomenon

The convergence is not uniform near a discontinuity. The partial sums exhibit an overshoot of approximately 9% of the jump size near a jump discontinuity, regardless of how many terms are included. This is known as the Gibbs phenomenon.

8.3 3. Smoothness and Convergence Rate

How fast \(S_N\) converges to \(f(x)\) is related to the decay rate of the coefficients \(a_n\) and \(b_n\), which in turn is related to the regularity (smoothness) of \(f\).

We have \(\|f - S_N\| \leq \sum_{|k|>N}(|a_k| + |b_k|)\), so faster decay of coefficients means faster convergence.

8.3.1 Case 1: Piecewise Continuous with a Jump

If \(f\) is piecewise continuous with a jump discontinuity:

\[|a_n|, |b_n| = O\!\left(\frac{1}{n}\right)\]

and the convergence is slow (the series won’t converge uniformly).

8.3.2 Case 2: \(f \in C^p\) (p-times Differentiable)

If \(f \in C^p([-\pi,\pi])\) with derivatives up to \(f^{(p)}\) continuous as a periodic function, and \(f^{(p)}\) is piecewise continuous (with at most finitely many jumps), then for some constant \(C\):

\[|a_n|,\, |b_n| \leq \frac{C}{n^p}\]

This implies:

\[\|f - S_N\| \leq C \cdot O\!\left(\frac{1}{N^p}\right)\]

Exercise: Prove this estimate by integration by parts \(p\) times.

8.3.3 Case 3: Smooth Functions (\(f \in C^\infty\))

If \(f\) is smooth (infinitely differentiable), then:

\[|a_n|, |b_n| = O(e^{-\alpha n}) \quad \text{for some constant } \alpha > 0\]

This gives exponential accuracy: the partial sums converge to \(f\) faster than any polynomial rate \(n^{-p}\).

8.4 4. The Discrete Fourier Transform (DFT)

8.4.1 Derivation

Computing Fourier coefficients requires estimating the integral:

\[c_k = \frac{1}{2\pi}\int_0^{2\pi} f(x) e^{-ikx}\, dx\]

Let \(x_j = jh\) where \(h = 2\pi/N\), and let \(f_j = f(x_j)\) for \(j = 0, 1, \ldots, N-1\).

Applying the composite trapezoidal rule:

\[c_k \approx \frac{1}{N}\sum_{j=0}^{N-1} f_j e^{-ikx_j} = \frac{1}{N}\sum_{j=0}^{N-1} f_j e^{-2\pi ijk/N} = \frac{F_k}{N}\]

The quantity \(F_k = \sum_{j=0}^{N-1} f_j e^{-2\pi ijk/N}\) is called the Discrete Fourier Transform of the sequence \([f_j]\).

8.4.2 Alternative Derivation via Discrete Inner Product

Define the discrete inner product:

\[\langle f, g \rangle_d = \sum_{j=0}^{N-1} f(x_j)\overline{g(x_j)}\]

The basis functions \(e^{ikx}\) are orthogonal under this discrete inner product:

\[\langle e^{ijx}, e^{ikx} \rangle_d = \begin{cases} 0 & \text{if } j \neq k \pmod{N} \\ N & \text{if } j = k \pmod{N} \end{cases}\]

So \(\{e^{ikx}\}\) is also a basis in the discrete inner product sense. For a function \(f\), we can find an approximation under \(\langle \cdot, \cdot \rangle_d\):

\[F_k = \langle f, e^{ikx} \rangle_d = \sum_{j=0}^{N-1} f_j e^{-ikx_j}\]

8.4.3 Definition

Definition: Suppose \(f(x)\) is a \(2\pi\)-periodic function. Let \(x_j = jh\), \(h = 2\pi/N\), and \(f_j = f(x_j)\).

The Discrete Fourier Transform (DFT) of the vector \([f_j]_{j=0}^{N-1}\) is the vector \([F_k]_{k=0}^{N-1}\) where:

\[F_k = \sum_{j=0}^{N-1} f_j\, W_N^{jk}, \quad W_N = e^{-2\pi i/N}\]

The Inverse DFT is:

\[f_j = \frac{1}{N}\sum_{k=0}^{N-1} F_k\, W_N^{-jk}\]

The DFT can be represented as matrix multiplication \(F = W\mathbf{f}\), where \(W_{jk} = W_N^{jk}\).

8.4.4 Proof of Inverse DFT

To verify the inverse formula, substitute and use orthogonality:

\[\frac{1}{N}\sum_{k=0}^{N-1} F_k W_N^{-jk} = \frac{1}{N}\sum_{k=0}^{N-1}\left(\sum_{m=0}^{N-1}f_m W_N^{mk}\right)W_N^{-jk} = \sum_{m=0}^{N-1} f_m \left(\frac{1}{N}\sum_{k=0}^{N-1} W_N^{(m-j)k}\right) = f_j\]

since \(\frac{1}{N}\sum_{k=0}^{N-1} W_N^{(m-j)k} = \delta_{mj}\) (equals 1 only when \(m = j\), and 0 otherwise).

Remark: The DFT is cyclic: the indices are taken modulo \(N\).

8.5 5. Aliasing

Consider \(f(x) = \cos(2x) + 2\sin(4x)\), which has frequencies 2 and 4.

When we compute the DFT with \(N = 16\) equally spaced samples, we can resolve all frequencies. If we shift the DFT output to be symmetric around 0 (i.e., using frequencies \(-N/2, \ldots, N/2-1\)), we recover the correct coefficients.

What if \(N\) is smaller? For example, \(N = 6\), the available frequencies are \(k = 0, \pm 1, \pm 2, \pm 3\). When we want frequency 4, there are not enough samples to resolve it.

The high-frequency components get aliased: they appear as low-frequency components in the sampled data. This effect is called aliasing or undersampling.

The high-frequency part will be aliased when we try to sample a signal with a frequency greater than the Nyquist frequency \(N/2\).

8.6 6. Trigonometric Interpolation

Definition: A trigonometric polynomial of degree \(d\) is a polynomial in \(e^{ix}\) of degree \(d\), i.e., a linear combination of \(\{e^{ikx} : |k| \leq d\}\).

For example, \(2\cos(x) + \sin(2x)\) is a trigonometric polynomial of degree 2.

8.6.1 DFT and Trigonometric Interpolation

The inverse DFT formula defines a function:

\[P_N(x) = \frac{1}{N}\sum_{k=0}^{N-1} F_k\, e^{ikx}\]

This is a trigonometric polynomial in \(x\), and satisfies:

\[P_N(x_j) = f(x_j) \quad \text{for } j = 0, 1, \ldots, N-1\]

Thus \(P_N\) is a trigonometric interpolant for the data \((x_j, f_j)\).

Remark: Trigonometric interpolation is not always a good choice for general function interpolation, as the Gibbs phenomenon can degrade accuracy near discontinuities.

8.7 7. The Fast Fourier Transform (FFT)

8.7.1 Computational Cost of Naive DFT

The DFT formula \(F_k = \sum_{j=0}^{N-1} f_j W_N^{jk}\) requires \(O(N^2)\) operations (for each of \(N\) values of \(k\), we sum \(N\) terms).

8.7.2 FFT Algorithm (Cooley-Tukey)

Assume \(N = 2^M\) for some integer \(M\). Let \(W = e^{-2\pi i/N}\).

Split into even and odd indexed terms:

\[F_k = \sum_{j=0}^{N-1} f_j W^{jk} = \sum_{m=0}^{N/2-1} f_{2m} W^{2mk} + \sum_{m=0}^{N/2-1} f_{2m+1} W^{(2m+1)k}\]

Define:

\[S_k = \sum_{m=0}^{N/2-1} f_{2m}\, (W^2)^{mk} = \text{DFT of even-indexed elements } (f_0, f_2, \ldots, f_{N-2})\]

\[T_k = \sum_{m=0}^{N/2-1} f_{2m+1}\, (W^2)^{mk} = \text{DFT of odd-indexed elements } (f_1, f_3, \ldots, f_{N-1})\]

Note that \(W^2 = e^{-2\pi i/(N/2)}\), so \(S_k\) and \(T_k\) are DFTs of length \(N/2\). Then:

\[F_k = S_k + W^k T_k, \quad k = 0, 1, \ldots, N/2 - 1\] \[F_{k+N/2} = S_k - W^k T_k, \quad k = 0, 1, \ldots, N/2 - 1\]

8.7.3 Complexity

The DFT of length \(N\) requires: - Two DFTs of length \(N/2\) (for even and odd parts) - \(O(N)\) additional operations to combine

Applying this recursively:

\[T(N) = 2T(N/2) + O(N) \implies T(N) = O(N \log N)\]

8.7.4 Example

For \(N = 2^{12} = 4096\) points: - Direct DFT: requires \(\sim 7 \times 10^7\) flops - FFT: requires \(\sim 10^5\) flops - FFT is approximately 700 times faster

8.8 8. Applications of DFT

8.8.1 8.1 Convolution

The convolution of two \(2\pi\)-periodic functions \(f\) and \(g\) is:

\[(f * g)(x) = \int_0^{2\pi} f(x - y)g(y)\, dy\]

From Fourier analysis, the Fourier transform of a convolution is the product of transforms:

\[\widehat{f * g}(k) = \hat{f}(k) \cdot \hat{g}(k)\]

The same holds for the DFT. The discrete circular convolution of data vectors \(\mathbf{f} = (f_0, f_1, \ldots, f_{N-1})\) and \(\mathbf{g} = (g_0, g_1, \ldots, g_{N-1})\) sampled from periodic functions is:

\[(f * g)_j = \sum_{\ell=0}^{N-1} f_\ell\, g_{j-\ell}, \quad j = 0, 1, \ldots, N-1\]

where the indices are taken mod \(N\) (so the convolution “wraps around”).

Key theorem: If \([F_k] = \text{DFT}(\mathbf{f})\) and \([G_k] = \text{DFT}(\mathbf{g})\), then:

\[\text{DFT}(f * g)_k = N \cdot F_k \cdot G_k\]

Proof:

\[\text{DFT}(f*g)_j = \sum_j (f*g)_j W^{jk} = \sum_j \sum_\ell f_\ell g_{j-\ell} W^{jk} = N F_k G_k\]

This provides an efficient way to compute convolutions: instead of \(O(N^2)\) direct computation, use FFT to compute DFTs in \(O(N \log N)\), multiply pointwise in \(O(N)\), then apply inverse FFT.

Remark: There are different conventions for the DFT across different fields. In this course we use:

\[F_k = \sum_{j=0}^{N-1} f_j e^{-2\pi ijk/N}, \quad f_j = \frac{1}{N}\sum_{k=0}^{N-1} F_k e^{2\pi ijk/N}\]

8.8.2 8.2 Spectral Accuracy of the Trapezoidal Rule

Revisiting the trapezoidal rule: Consider the integral \(I = \int_0^{2\pi} f(x)\, dx\) approximated using \(N\) equally spaced points \(x_0, \ldots, x_{N-1}\), \(h = 2\pi/N\):

\[T_h[f] = h\sum_{j=0}^{N-1} f(x_j)\]

Consider \(f(x) = e^{ikx}\) on \([0, 2\pi]\). The exact integral is:

\[I = \int_0^{2\pi} e^{ikx}\, dx = \begin{cases} 2\pi & k = 0 \\ 0 & k \neq 0 \end{cases}\]

The trapezoidal approximation is:

\[T_h[e^{ikx}] = h\sum_{j=0}^{N-1} e^{ikjh} = \begin{cases} 2\pi & k \equiv 0 \pmod{N} \\ 0 & \text{otherwise} \end{cases}\]

So the trapezoidal rule is exact for \(e^{ikx}\) unless \(k\) is a multiple of \(N\).

Consequence: Suppose \(f(x)\) has Fourier series \(f(x) = \sum_k c_k e^{ikx}\). Each point added to the trapezoidal rule improves the accuracy by capturing one more term in the Fourier series. The trapezoidal rule inherits the same convergence rate as the Fourier series:

\[\|T_h[f] - I\| = O(h^p) \quad \text{for any } p, \text{ if } f \in C^\infty\]

8.8.3 8.3 Accuracy of Trigonometric Interpolation

Recall the trigonometric interpolant \(P_N(x) = \frac{1}{N}\sum_k F_k e^{ikx}\).

The approximation \(F_k/N\) differs from the true Fourier coefficient \(c_k\) by a trapezoidal rule error. Thus:

\[\|P_N - S_N\| \leq \text{trapezoidal rule error}\]

Since both \(\|f - S_N\|\) and \(\|P_N - S_N\|\) are related to the smoothness of \(f\):

\[\|f - P_N\| \leq \|f - S_N\| + \|S_N - P_N\|\]

is also related to the smoothness of \(f\). The interpolant achieves the same spectral accuracy as the Fourier partial sum.

8.8.4 8.4 Discrete Cosine Transform (DCT)

For a real-valued function on \([0, \pi]\), it is more convenient to use only cosine functions to define the transform and interpolation (a “cutoff” version of the Fourier transform).

Setup: Consider \(f(x)\) real-valued on \([0, \pi]\), given at \(N\) points \(x_j = j\pi/(N-1)\), \(j = 0, 1, \ldots, N-1\).

Even extension: Extend the data symmetrically:

\[\tilde{f}(x_j) = f_j, \quad \tilde{f}(x_{2N-2-j}) = f_j, \quad j = 0, 1, \ldots, N-1\]

This creates \(2(N-1)\) points on \([0, 2\pi)\) with the even extension satisfying \(f_0 = f_{2N-2}\).

Compute the DFT of the extended data \([\tilde{f}_j]\). Using the even-extension symmetry:

\[e^{-ikx_{2N-2-j}} + e^{-ikx_j} = 2\cos(kx_j)\]

so the DFT reduces to:

\[F_k = \sum_{j=0}^{N-1} \tilde{f}_j e^{-ikx_j} \longrightarrow \tilde{F}_k = \sum_{j=0}^{N-1} f_j \cos(kx_j)\]

This is the Discrete Cosine Transform (DCT), which involves only real arithmetic.

8.8.5 8.5 Discrete Derivatives via DFT

Suppose we have data \([f_j]\) for a real-valued function \(f\) on \([a, b]\). Recall that numerical approximations to derivatives can be derived by taking derivatives of an interpolant.

Key idea: For periodic \(f(x)\) on \([0, 2\pi]\), use DFT to compute derivatives spectrally.

If \(f(x) = \sum_k c_k e^{ikx}\), then:

\[f'(x) = \sum_k (ik)\, c_k\, e^{ikx}\]

Differentiation in \(x\) corresponds to multiplication by \(ik\) in frequency domain.

Algorithm for spectral differentiation (assuming \(N = 2m\)):

  1. Compute \([F_k] = \text{DFT}([f_j])\)
  2. Multiply: \(\hat{F}_k = (ik) F_k\) for \(k = -m+1, \ldots, m-1\), and \(\hat{F}_{\pm m} = 0\) (the Nyquist mode is set to zero)
  3. Compute \([f'_j] = \text{IDFT}([\hat{F}_k])\)

In other words, the approximate derivative at the grid points is:

\[f'(x_j) \approx \frac{1}{N}\sum_{k} (ik) F_k\, e^{ikx_j}\]

Comparison with finite differences:

Method Cost Accuracy
Finite difference (order \(p\)) \(O(N)\) \(O(h^p)\)
Spectral (DFT-based) \(O(N \log N)\) Exponential (for smooth \(f\))

The spectral method achieves spectral accuracy: exponentially convergent for smooth periodic functions, at the cost of \(O(N \log N)\) operations using the FFT.

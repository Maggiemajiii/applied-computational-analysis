# Applied Computational Analysis Course Notes

This repository contains course notes for Applied Computational Analysis, organized as a bookdown project.

## Structure

- **Lecture Notes**: Notes from lectures 1-3 are available in the `lecture-notes/` directory
- **Course Content**: Organized by lecture and topic in R Markdown files
- **Output**: HTML book is generated in the `docs/` directory

## Building the Book

To build the book, use:

```r
bookdown::render_book()
```

The output will be generated in the `docs/` directory.

## Contents

- Lecture notes from the course
- Additional mathematical topics and supplementary materials
- Bibliography and references
